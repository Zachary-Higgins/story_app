## [Unreleased]
## [1.0.0] - 2026-01-18

### Changed
- Release 1.0.0


### Added
- Dual-mode build system: local app + npm package
- Vite plugin for automatic story discovery from `content/stories/*.json`
- Story authoring with multiple layout types (hero, split, timeline, immersive)
- Theme system with CSS variables
- Audio controller for background music
- Scroll progress indicator
- Complete TypeScript support with full type definitions
- Comprehensive test coverage (33 tests)
- ESLint + TypeScript linting
- Integration guide and story authoring documentation

### Changed
- Theme selection is now author-controlled (removed from story page UI)
- Story index auto-generated by plugin (no manual editing required)

### Fixed
- CSS bundling for npm package consumers
- Plugin module resolution when imported from `node_modules`

## [0.0.1] - 2026-01-18

### Added
- Initial release
- React 18 + Vite 7 + TypeScript foundation
- Story Engine component with content-driven architecture
- Vite plugin for story discovery
- Full documentation and agent playbooks
